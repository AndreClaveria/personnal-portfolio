# Portfolio en ligne

## Idée du portfolio

### Un portfolio pixel art à plusieurs pages, au moins 5 minimum :

    - Ecran du début  : Une ville de nuit éclairé par une lumière violette
    - Accueil : Appartement qui contient une télé, une fenêtre et une bibliothèque
    - Présentation : Entrant dans la télé, on peut changé de chaîne pour ainsi voir plusieurs facette de moi
    - Expériences : On sort par la fenêtre et on voit un sorte de panneau publicitaire qui s'éclaire lorsqu'on scrolle ou qu'on active les lumières.
    - Projets : On va dans la bibliothèque et lorsqu'on prend un livre ça ira vers un de mes projets.

## Outils

    - Krita
    - VSCode : NextJS, SCSS, NodeJS
    - Figma
